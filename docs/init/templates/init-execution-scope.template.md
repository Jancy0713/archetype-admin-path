# Init Execution Scope Template

这份模板用于生成第8步执行使用的初始化范围文档。

目标：

1. 只描述第8步允许执行的纯工程初始化内容。
2. 不写任何具体业务接口、业务 contract、真实业务页面实现。
3. 每条内容都写成可检查的清单项，方便执行与 reviewer 校验。

建议结构：

## 1. 文档定位

- 说明这是第8步执行范围文件。
- 说明它只约束初始化工程，不替代后续 PRD。

## 2. 允许执行

- 逐条列出第8步允许做的事情。
- 每一条尽量对应真实代码或文件动作。

## 3. 明确不做

- 逐条列出本阶段禁止做的事情。
- 尤其明确哪些内容必须留给 PRD / contract。

## 4. 必须交付

- 逐条列出第8步结束时必须已有的工程产物。
- 尽量指向代码、目录、文档或规则文件。

## 5. 完成标准

- 逐条列出达到“第8步完成”的判断条件。
- 这些条件要能让 reviewer 直接检查。

## 6. reviewer 检查点

- 明确 reviewer 应重点检查的风险边界。
